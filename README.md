# Alfred

Personal terminal assistant for all operating systems and languages

## Install

Install requeriments

```
pip install -y halo
pip install --upgrade -y openai
```

Clone repository

## Usage

You just need to ask a question and **alfred** will answer it. To finish type `exit`
